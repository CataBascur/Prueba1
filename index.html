Sobre mí

Soy educadora de párvulos titulada con mención TEL (trastornos específicos del lenguaje), amo mi rubro y me dedico actualmete a esto. Hace poco se me hizo interesante el desarrollo web, por lo que ahora soy estudiante de Desarrollo Full stack javascript. Mi deseo es poder poder ser un aporte en la educación a través de este nuevo conociemiento.

Portafolio

¡Hola! Aquí puedes ver los proyectos en los que he trabajado hasta ahora. Te comento una breve descripción de lo que se hizo en cada proyecto y al lado izquierdo puedes ver una imagen de lo que se realizó.

Razas de conejos

Web ficticia creada para incorporar los primeros conocimientos que adquirí en HTML y CSS. El proyecto emula una guía informativa de las distintas razas de conejos, motivando a adoptar.

Iguana Page

Web ficticia creada para incorporar adquiridos en HTML, CSS y Flexbox. El proyecto representa un tipo de wiki donde se exponen las principales características de este animal y fotografías. Además se utiliza por primera vez íconos de Font Awesome.

Cuppon

Web ficticia creada para incorporar adquiridos en HTML, CSS, Flexbox y Bootstrap. El proyecto presenta una página web donde se venden cupones para distintos servicios. El proyecto fue realizado de manera responsiva, para visualizarlo en formato web de escritorio y mobile.
